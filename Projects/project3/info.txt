Title - GenAI Intelligent Email Triage & Jira Automation
Description - Smart automation using GenAI to analyze incoming project emails and conditionally create Jira tasks only when actionable work is detected.
Detailed - <b>Project Overview:</b><br>Designed an intelligent inbox assistant to filter noise and capture action items. Unlike standard automations, this solution uses GenAI to read emails and make logical decisions on whether a task exists.<br><br><b>The Logic Flow:</b><ul><li><b>Intent Analysis:</b> Ingests project emails via Outlook and prompts the GenAI API to summarize the content and determine if a development task is required.</li><li><b>Manager Notification:</b> Instantly pushes the AI-generated summary to the Project Manager via MS Teams for visibility.</li><li><b>Conditional Logic Gate:</b> The workflow evaluates the decision flag from the AI. <b>If (and only if) a task is required</b>, it triggers the Jira creation process.</li><li><b>Action Execution:</b> Automatically logs the ticket in Jira with a pre-filled description and summary derived from the email analysis.</li></ul><p><b>Tools:</b> Power Automate, OpenAI API, Jira, MS Teams.</p>
