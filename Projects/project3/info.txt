Title - GenAI Automated Project Reporting Ecosystem
Description - End-to-end automation pipeline using Power Automate and GenAI to transform raw Jira data into strategic Power BI dashboards.
Detailed - <b>Project Overview:</b><br>Designed a fully automated reporting loop to eliminate manual data extraction from Jira. The solution integrates Generative AI to interpret raw project data and visualize it instantly.<br><br><b>The Automation Workflow:</b><ul><li><b>Intelligent Extraction:</b> Triggered by Jira subscription emails, the flow uses a GenAI API to parse unstructured text and extract key project metrics into a structured table.</li><li><b>Data Integration:</b> Automatically archives processed data into a SharePoint list, serving as a historic repository.</li><li><b>Live Visualization:</b> Triggers a Power BI dataset refresh immediately after data ingestion to ensure real-time accuracy.</li><li><b>Stakeholder Communication:</b> Dispatches a final status update via MS Teams to keep leadership informed.</li></ul><p><b>Tech Stack:</b> Power Automate, OpenAI API, Jira, SharePoint, Power BI, MS Teams.</p>
