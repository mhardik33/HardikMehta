Title - Automated Global Update Sharing
Description - Enterprise-wide communication automation that distributes critical updates and visualizes engagement metrics in real-time using Power BI.
Detailed - <b>Project Overview:</b><br>Designed a unified communication pipeline to streamline how cross-functional updates are shared. This solution replaces manual email chains with an automated distribution system and a live analytics dashboard.<br><br><b>The Solution Architecture:</b><ul><li><b>Centralized Intake:</b> Uses a standardized Microsoft Form to collect updates from global stakeholders, ensuring consistent data quality.</li><li><b>Automated Routing:</b> A Power Automate flow processes new submissions, formats the content, and dynamically routes notifications to the relevant Teams channels based on the update category.</li><li><b>Impact Analytics:</b> Simultaneously feeds data into a Power BI dashboard, visualizing key metrics like update frequency by game area, milestone progress, and monthly trends.</li><li><b>Operational Gain:</b> Achieved a 60% improvement in communication efficiency and ensured zero critical updates were missed during development cycles.</li></ul><p><b>Tech Stack:</b> Power Automate, Microsoft Forms, Power BI, MS Teams.</p>
